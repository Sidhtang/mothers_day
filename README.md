# Happy Mother's Day, Maa 🩷

A heartwarming website built with Next.js — ready to deploy on Vercel in seconds.

## Deploy on Vercel (easiest way)

1. Push this folder to a GitHub repo
2. Go to [vercel.com](https://vercel.com) → "Add New Project"
3. Import your GitHub repo
4. Click **Deploy** — that's it! Vercel auto-detects Next.js.

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Project Structure

```
app/

  layout.tsx      ← Root layout + metadata
  globals.css     ← All styles (fonts, animations, layout)
public/
```
